# Material UI Tutorial
React에서 사용할 수 있는 라이브러리인 Material UI을 실습하는 애플리케이션

## 사용 기술
- Front-End : React
- Back-End : 없음
- Database : 없음
- Deployed : Heroku

## Start producton mode
1. 레포지토리의 소스코드 pull
2. npm i
3. npm start
4. localhost:3000에서 확인
5. 만약 client를 수정했다면 npm run build 선행 필요

## Start development mode
1. 레포지토리의 소스코드 pull
2. npm i
3. npm run dev
4. localhost:3000에서 확인
